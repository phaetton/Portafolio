aria-label Aria-label es un atributo que puede utilizar para proporcionar una etiqueta breve y concisa para un elemento que no tiene un contenido de texto visible. Por ejemplo, puede usar aria-label para etiquetar un botón que solo tiene un icono o un enlace que solo tiene una imagen. Aria-label anula cualquier otro contenido de texto o texto alternativo que pueda tener el elemento. El valor de aria-label debe ser una cadena que tenga sentido para el usuario y describa el propósito o la función del elemento.
